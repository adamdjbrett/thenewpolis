### HOW TO

+ Install Wordpress on your host
+ Upload our Wodam Wordpress Headless cms on your WP-themes and activate
+ Next you can access on 11ty Config menu into your backend WordPress CMS and update it - we convert  eleventy metadata.js into wodam 11ty config.
+ Create a new repo project on gihtub
+ Push your 11ty project
+ Open on `_data/wordpress.js` and configure `CONST BASE URL` with your WP backend url
+ Deploy
+ Congratulations you have run a WP as Headless CMS feat 11ty JAMSTACK
+ Simply you can write blog post or create new static page on default WP menu
+ You can update meta tag, and others widget into our Wodam Backend CMS.
